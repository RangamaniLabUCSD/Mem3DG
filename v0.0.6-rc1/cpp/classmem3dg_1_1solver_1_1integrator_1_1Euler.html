Detailed Description

Euler (gradient descent) time Integration.

Parameters
system_System object to be integrated
characteristicTimeStep_time step, or the initial time step for backtracking algorithm
totalTime_total simulation time
savePeriod_period of saving output data (in the unit of characteristicTimeStep_)
tolerance_tolerance of force L2 norm for termination
outputDirectory_path to the output directory
frame_frame index, if nonzero, enable continuation mode
